WordPress只能支援PHP主機-噗浪討論

by元兒～ 2010-08-03 未整理舊文

有人在噗浪上問說能不能把WordPress架在ASP主機上(我猜主機應該是用IIS架的)，我最近也正好遇到類似的問題(想用Xoops架在別人的IIS主機上)，今天在噗浪上討論了一些：

小荷花說 WordPress只能支援PHP主機、不能放在asp，請問這種問題可以破解嗎???我的問題是「有沒有讓WordPress放在asp主機的方法?」謝謝知道方法回覆我的朋友們^^
Redd說 ASP主機可安裝 php
小荷花說 Redd: 我想做一個WordPress部落格，但是銀行的主機是asp的，所以不知道做好了要怎麼放上去
小荷花說 Redd: ASP主機可安裝 php --今天資訊室的人出差，我再問問看他們是否知道。謝謝
Redd說 ASP主機可安裝 php，就可支援 php 程式，不過如果是銀行的主機，資訊室為了省事，可能會用什麼安全問題之類的理由，拒絕你…
元兒～說 我也遇到這個問題，最近找到替代的-PJBlog，是用ASP寫的，不過我還沒試過～
Redd說 我之前也是自己架設部落格 www.euseless.com/myblog/
Redd說 後來覺得太麻煩了，還是搬到無名
Redd說 我的無名 www.wretch.cc/blog/reddliaw
Redd說 所以要自己架設的話 … 請三思~
小荷花說 謝謝你們的回覆，因為一堆金融法條關係，之前那些經營了兩年多的部落格都得被迫關閉…….。現在在找asp主機可以支援的部落格，我們資訊室的人又超忙超難找Orz
小荷花說 因為WordPress做出來的部落格真的好漂亮，而且網路上比較容易搜尋到WordPress疑難雜症的解決方式，所以才想架他們家的
小荷花說 Redd: 你意思是說，將部落格放在asp是一件很麻煩的事情嗎?
阿利 GoGo說 是php架構的部落格要放在asp上面是一個麻煩的事情~~~
小荷花說 阿利 GoGo: 釗利~拜託簡述幾點有多麻煩~~~~~Please~~~~
元兒～說 WordPress.com 不然也可以直接用線上版的WordPress，直接註冊帳號密碼就行了，不過限制比較多，可以參考不過不建議使用！線上版網址後面是com，不是org喔！
元兒～說 我也曾用Win7的IIS架站，光是要讓IIS支援PHP就有夠麻煩，要下載以外，還要設定什麼MIEE的，到最後仍失敗收場～ 如果要架PHP站的話，建議是用appache架，與PHP配合度較佳 。不過如果是在別的地方架，那和資訊組……得好好溝通吧！
小荷花說 元兒～: 請問~”不建議使用”是大概基於什麼原因
小荷花說 我先”暫時”投降各位….這問題已經燒光我所有腦細胞了，因為連你們這樣專業有實做經驗的都告誡我有點麻煩，我還是先暫緩這計畫…感謝你們的經驗分享^____^
元兒～說 就是線上版的WordPress和架站版的街面雖然相似，也有後台管理介面，但功能差很多，沒有離線版的那麼自由，尤其是不能編輯CSS，也只能用現成佈景主題，不能擴充！還有部分JavaScript會受到限制……而架站版的就沒有這些限制，但前提還是要把PHP的問題搞定！
小荷花說 我的用意是在 “製作一個部落格形式(可以讓我隨時編輯、新增文章)”，而且”可以放在asp”主機，為何要堅持放在asp主機?沒辦法..因為前兩週”金管會”管到我們這裡來了Orz，我做出來的東西必須是可以讓主管隨時監控的啊~
小荷花說 元兒～: 我整理一下您說的，線上版的WordPress、有後台管理介面、不能編輯CSS..但是，如果只是做單純文字文章編輯新增+可以放在ASP主機，這樣..對吧?
小荷花說 元兒～: 還有喔~我們全公司上下包括我，都是用WIN XP作業系統運作
元兒～覺得 ……我不是這個意思～我說的線上版是直接用WordPress的主機，不是架在自己的主機上，就像是無名、痞客幫、Blogger，只要註冊自己的帳號密碼就行了，圖片也是用他的網路空間！
小荷花說 元兒～: 喔………………不好意思我誤會意思>” 好，那這樣的話我就無法使用。結論是這樣Orz
小荷花說 元兒～: 我剛剛估狗了這串字”支援ASP的部落格”，找到這篇文章 國內幾家支援 asp.net 的虛擬主機。請問您，asp主機還有分 net 1.1 或2.0這樣?謝謝..
小荷花說 唉..我現在想要執行的東西因為權責問題全部不歸資訊室管，在管理主機問題的反而是一問三不知的非專業人士，所有問題都只能透過GOOGLE或是網路達人協助了Orz
元兒～ 哈哈哈！考倒我了其實我沒有真的自己架過站，頂多是稍微隨便架遺下，是用來測試ASP、PHP而已～至於你說的net 1.1 或2.0我就不懂了～
小荷花說 元兒～: ok謝謝^^
元兒～ 還有如果負責管理主機的人是無法解決PHP的問題，那就沒辦法了～因為會牽扯到更動主機的設定、外掛……一定得要先想辦法讓主機支援PHP，不然WordPress的是什麼都不用談！
小荷花說 元兒～: 對啊，找了三天的資訊加上你們提供的經驗分享，我得到的結論也是如此，我們那個秘密單位= = 一向只負責監督管理，不負責解決問題，所以都得靠自己了。
元兒～ 我也遇到這個問題，Xoops和WordPress也是用PHP寫的，想架在別人的IIS主機上，真的很麻煩，要讓IIS支援PHP……光是在我自己架就遇到瓶頸。有人建議改用Appache架，但總不可能叫他們就真的換吧……
元兒～ 後來看到別人在同一個主機弄成功的，不是用Xoops，也不是用WordPress，而是用ASP寫出來的PJBlog，不過我不知道是什麼，還得研究看看……
小荷花說 元兒～: okok，這資訊多少對我有點幫助，謝謝..我也努力搜尋看看^^
