網路流量監控、網路限速

by元兒～ 2010-09-09 未整理舊文

我現在人在大學宿舍，大進宿舍前幾天，我就已經準備好一些軟體準備搬去宿舍用～我就稍微分享一下！

流量監控-BitMeter

下載點：官網 http://codebox.org.uk/pages/bitmeter2（安裝後就有中文介面了）

有些大學宿舍網路會限制你的流量，像是某人讀的科技大學，一天上傳下載量不能超過500MB，否則會停用一個禮拜……有些學校是規定速度不能超過多少M。

如果你擔心你自己的流量有沒有超過學校的限制，那可以在電腦上安裝這個軟體，讓軟體幫你監控你的流量，而且還可以設定警示，如果快要超過學校規定的流量，這軟體就會警告你喔！

  • 在圖表上按右鍵→「統計格線」→就能看到本日、本周、本月的總流量和平均速率
  • 如果宿舍有上傳下載量限制的話，在圖表上按右鍵→「ISP警示」→然後自行調整就好了～！

網路限速-NetBalancer

官網免費版下載：http://www.seriousbit.com/download?file=NetBalancerSetup.exe

像我的宿舍網路頻寬是兩間共用10M，一間有5個人。如果在共用網路的情況下，有人用網路用得很兇（像是跑p2p下載、看網路影片的……）很可能會因為一個人把共用的10M速度全部佔走，別人上網的速度就會慢得像烏龜，而造成別人的困擾……

這個軟體就是限制你自己的網路速度要到哪個程度，才不會把所有頻寬被你佔掉而造成別人的困擾喔～

PS.我在大學宿舍是使用PPPoE寬頻連線，NetBalancer好像對PPPoE起不了限速作用，不曉得其他人有沒有這個問題～

網路電視-在宿舍也能看電視

by元兒～ 2010-09-07 未整理舊文

在家裡，「電視」是個非常舒服的娛樂～但是在大學宿舍，可是沒有電視的！就算有，大概就是交誼廳的電視，是大家共用的，當然不能隨自己的意願轉台～我再來要住在宿舍，當然忘不了這點啦～（我這個死小孩）

我已經下載了兩個網路電視軟體，都是我很推薦的！是我再來要在大學宿舍用的！不過話先說在前頭～網路電視很不穩定，由時候會lag很嚴重，甚至會有一些電視台不能看，而且因為是網路電視，可以看到一些國外的頻道，但台灣頻道並不是每個都有！

PPStream-影音資料庫豐富

下載：阿榮福利味 http://azo-freeware.blogspot.com/2006/10/ppstream-1001702.html
下載：官網 http://www.ppstream.com/download.html

 

收錄了非常豐富的電視節目資源，讓你選擇你要看什麼節目。光是動漫卡通就已經有三萬多個，而且幾乎每一集都有！除了動漫卡通以外，還收錄了電視劇、電影、綜藝、體育……是我蠻喜歡的一套軟體～！不過美中不足的是這是大陸軟體，而且有廣告……

TVAnts-線上電視直播

下載：官網 http://www.tvants.com/index.html

是真正電視直播的！不是讓你選節目，而是選電視頻道。選擇頻道以後，就是這個電視頻道正在撥放的東西！和一般電視轉台差不多！

題外話：還有一個是在大學宿舍防無聊的看網路漫畫軟體-Comic Shelf，可參考我的網誌

Win7「還原舊版」……救了我一命～

by元兒～ 2010-09-01 未整理舊文

我今天在工作時，好像是因為程式出錯而導致下面內容莫名其妙消失，而我沒注意到就把目前的文件覆蓋儲存下來，等到我發現時，檔案早已被覆蓋，而且軟體本身的復原次數已達上限……難道消失的內容還得又要自己重建嗎？

在我自己快瘋掉時，突然想到Windows7檔案總管理有個「還原舊版」功能，就趕快試試，阿彌陀佛阿～再覆蓋之前的檔案終於救回來了阿～

使用方法：

在你已經被覆蓋需要還原的檔案上按右鍵→選擇「還原舊版」

→然後把你需要哪個時間點的檔案拖出去就行了！（也可以選擇以後按下面這三個鈕）
　「還原舊版」之神啊～你救了我啊～～～我要尊敬你～～～
 

題外話：如果要做網頁，盡量不要是從MS Word轉換過來的！轉換過來的HTML標記非常亂，再搭配其他網頁設計軟體，很可能會使軟體錯亂！

Firefox 4.0 beta4測試報告

by元兒～ 2010-08-28 未整理舊文

Firefox4的測試版已經出來了！我已經搶先下載安裝了！因為現在還在測試階段，所以現在到官網首頁也只看到Firefox3.6～我原本已經安裝了Firefox3.6，安裝Firefox4後沒有把先前的Firefox3.6取代掉，所以我的電腦現在有兩個Firefox並存！

Firefox 4.0 Beta4：http://www.mozilla.com/pt-BR/firefox/all-beta.html

Firefox4整個介面大改版了，以符合瀏覽器介面的趨勢，讓完全沒碰過瀏覽器的人能快速上手！而這次的Firefox4除了介面大變以外，速度上也是這次主要的改版目標！我覺得Firefox4在速度上有大幅提升喔！

 

在Beta4加入了「分頁群組」功能，而且有很直覺化的「圖形化桌面空間」設計，在分頁管理畫面，能直接用滑鼠拖曳來指定分頁群組要怎麼樣。我看網頁都常常開了一大堆，而分頁變得很雜亂，我正好需要這個功能！

有全新的附加元件管理介面，整個介面看起來舒爽許多……因為Firefox4還只有測試版，許多擴充套件還沒辦法與Firefox4相容～

不過我發現目前有個缺點，就是我使用Personas佈景主題時，右上角的Windows視窗那三個按鈕消失了，沒有叉叉、放到最大、縮到Win工具列按鈕，這可能還是個Bug吧……

我聽說接下來的Beta5還會有重大改變，就拭目以待吧～

Firefox偽裝成其他瀏覽器（含手機）-User Agent Switcher

by元兒～ 2010-08-28 未整理舊文

之前有人在網路上介紹Blogger Touch給我，使可以讓Blogger部落格也擁有手機專用的網頁喔！我使用這個以後會得到CODE，這個CODE會偵測這台電腦是不是手機，因為我沒有智慧型手機，而在網路上找到變通的測試工具-User Agent Switcher。

User Agent Switcher https://addons.mozilla.org/zh-TW/firefox/addon/59/

 

通常擁有手機板的網頁，會抓取瀏覽器資料，來判斷是不是用手機上網。而User Agent Switcher就是對這個網頁修改瀏覽器的資訊，可以讓網頁誤以為你是用iPhone、Android……上網，就把你帶到為iPhone、Android……量身訂做的網頁喔！如果不想偽裝了，只要在User Agent Switcher選單上選擇「Default User Agent」就行了！

也可以自訂這些偽裝的資訊喔！或是直接下載匯入別人已經整理好的user agent xml 檔喔！

俗稱的上帝模式……啟動你的Windows 7無敵控制台

by元兒～ 2010-08-20 未整理舊文

其實我一聽到這個詞還覺得蠻好笑的，我還「無敵模式」……

這是我在pcUser的「Windows 7 頂級重灌術」一書學到的密技～大家都知道有Windows有控制台這個東東吧～在Win7裡還隱藏了更厲害的控制台-「上帝模式」（網友俗稱！微軟官方不是這樣叫的）裡面蒐錄了276個設定捷徑喔！
警告：這是在Windows 7使用的，拿到其他作業系統使用可能會造成無法正常開機喔！

使用方法：

隨便在一個地方新增資料夾，這個資料夾的名字要取：
「工程模式.{ED7BA470-8E54-465E-825C-99712043E01C}」（藍色字是你要取什麼名字隨你高興）建完後你會發現原本後面這一長串字消失了，而圖示變成控制台的圖示，看到這種現象就表示成功了，趕快點進去看看效果吧！

Wot 自動偵測網站安不安全

by元兒～ 2010-08-20 未整理舊文

大家都很常上網吧～瀏覽網頁最怕的就是遇到惡意網站，如果防毒軟體有擋掉的話就還好，如果防毒軟體沒擋到，自己又不知道這是惡意網站，那後果就……慘～

我很推薦Wot這個瀏覽器外掛，也希望大家安裝這個東東。Wot是個免費自動檢測網頁安全性工具，主要是依賴社群來評等！目前有繁體中文，支援Firefox、Chrome、Internet Explorer、Safari。

►網路安全性 | WOT Web of Trust http://www.mywot.com/zh-hant

安裝後瀏覽器會多出Wot按鈕，這個按鈕會根據這個網頁是否安全而改變顏色，如果是深綠色，那這個網頁受到很多人的信任，也就越安全。如果瀏覽到壞網站時，還會跳出大大的警告！

在搜尋網站時，網頁項目旁邊也會出現wot圖示，可以讓使用者在進入之前就能先知道這個網站受不受信任。

當然自己也可以在上面評等，按下Wot按鈕就能評等了。多了一個人評等，就能讓Wot更準確喔！

電腦玩物-WOT 上網搜尋瀏覽的免費避雷器，社群力量共建我們的信任網路有更詳細的介紹喔！

作業系統太新 老舊軟體不能用？

by元兒～ 2010-08-19 未整理舊文

昨天晚上有人來我們家，是為了請教我爸一些問題。我爸要配合專業軟體講解時，遇到麻煩的問題，就是這台電腦的作業系統太新了，導致與這個軟體不相容而無法使用～但又非得使用這個軟體……

昨天我用了以下方法成功解決相容性問題，主要是讓電腦模擬舊版作業系統的環境開啟：

先在要開啟老舊程式的執行檔案右鍵→選擇「傳送到」→「桌面(建立捷徑)」
（如果不是在光碟裡的話，也可以選擇「建立捷徑」）
（PS. 本擷圖是我寫部落格的舉例，並非昨天當時情況）

→然後在剛剛建出來捷徑檔案上按右鍵選「內容」

→在相容性上勾選「以相容模式執行這個程式」，並調成Windows XP以下版本（建議）。如果這個程式有出現權限不足問題的話，那也勾選「以系統管理員的身分執行此程式」吧！調完按確定以後，趕快在捷徑上點兩下開啟看看吧！說不定就能開了！

注意！本方法只是”暫時”解決不能開啟的問題，畢竟模擬出來的環境和真實環境還是有差！用這個方式強開程式，就算成功開，很可能在使用途中還會遇到很多狀況（中途當機、不相容……），而且不保證所有的程式都能用這個方法開！

我建議還是去找找這個軟體有沒有新版的（通常新版應該會支援Windows 7）有的話是最好，沒有的話就用這個方法看看！

Kylo-針對影音網站設計的瀏覽器

by元兒～ 2010-08-10 未整理舊文

相信大家都知道Windows Media Center吧！是能把電腦融入家庭客廳，成為客廳的影音系統！整個程式介面設計都是針對電視，當然就是要大字體，畫面要精緻、舒適啦！不過我這次不是要介紹這個，而是針對影音網站設計的瀏覽器-Kylo！

Kylo本身是個瀏覽器，是使用Firefox的Gecko核心，一樣有分頁瀏覽、書籤功能、可以列印、下載檔案，可以說是電視版的Firefox，但不能使用擴充套件，介面上可以說是相似又不相似！而且還可以叫出螢幕鍵盤輸入，不過目前沒有中文，但仍可以按Ctrl+空白鍵切換輸入法到中文。
一進去就會出現像下面的畫面，可以選擇你想到哪個影音網站，如果沒有在列表裡，也可以直接輸入下方的網址列！當然也少不了Youtube啦！而且Youtube（如上圖）有針對這個軟體設計的介面喔！

如果電腦裡有Windows Media Center的話，在安裝時還可以直接在Media Center加入Kylo程式捷徑喔！

不過並不是列表裡所有的網站都有特製介面，Facebook就沒有！點進去後仍然是普通版網頁……

更多介紹、安裝說明可以參考免費資源網路社群-Kylo 專為網路影片而設計的瀏覽器
