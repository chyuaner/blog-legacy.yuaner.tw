Windows關機小技巧

by元兒～ 2010-05-23 未整理舊文

在Windows裡關機，大家都知道要從開始功能表按下關機，也有人是直接按主機電源鍵關機（按一下是發出關機訊號，按久一點是強迫關機）（注意！有些舊機型的主機按一下電源鍵就是強迫關機，我阿公家就有），其實還有另一種關機法……（Windows XP、Vista、7適用喔～）

按「開始功能表」→「執行」→輸入「cmd」按「確定」
（ViSta、7使用者則是在開始功能表的搜尋框輸入cmd）

在「命令提示字元」裡輸入關機的指令：
如果我要在30秒後關機，那我就輸入「shutdown -s」，
那我想指定己XX秒後關機，就輸入「shutown -s -t XX」，
如果我要電腦OO秒後重新開機，輸入「shutdown -r -t OO」即可～

（第二張圖是因為我那邊有點怪怪的，就先用別人的電腦示範）
輸入後電腦就會出現準備關機的訊息，時間一到電腦就會關機了喔！！
如果已經輸入指令後反悔了，想終止，只要輸入「shutdown -a」就可以了！（不過要在倒數結束以前才有效）
有很多人碰過這種情況：如果莫名其妙出現這個訊息，可以先用這個指令終止，但是會莫名其妙出現，可能是這台電腦中毒了，這個指令只是暫時不讓電腦關機，並不能真正解決，還是得要清除病毒才行！！

還在用IE上網嗎？挑一挑更適合的吧！

by元兒～ 2010-03-08 未整理舊文

之前班上有人問我說要怎麼把Internet Explorer 6(簡稱IE6)升級到IE8，他說Youtube已經停止支援IE6了，必須要升級…
不過我回答說不要升級到IE8，因為現在有幾個更好用的網頁瀏覽器，速度、功能表現都比IE還要好。有人說他已經習慣IE了，但是從IE7開始，整個介面大改版，就算升級IE，也得要重新適應，而且之前還爆出IE有一連串的漏洞事件，讓很多人開始懷疑IE的安全性，甚至有其他國家政府發起「拒用IE」的指示，我覺得要升級IE然後在習慣新介面，還不如直接換個適合自己的瀏覽器。
其實我已經沒在用IE了，現在最常用Mozilla Firefox，而且還當成我預設的瀏覽器呢！
我在用的Mozilla Firefox、Google Chrome，目前都是評價非常高的網頁瀏覽器，這兩個瀏覽器各有利弊，這篇網誌希望能讓大家選擇對自己合適的網頁瀏覽器。

Google Chrome—對一般人來說是最佳選擇

對一般只想單純上網的話，我會強烈推薦Google Chrome！！
Chrome是個簡單到不行的程式，畫面不但簡潔，功能也非常單純，完全沒有雜七雜八不必要、多餘的功能，相信所有人都能很快、輕鬆上手！
目前Chrome是效能最好、速度最快又穩定的網頁瀏覽器，它把所有分頁、所有擴充程式都分開處理，所以如果有其中一個當掉，不至於影響整個瀏覽器的運作。之前電腦雜誌有做比較，Chrome在效能上還贏過Firefox呢！
目前Chrome已經有Linux版本了，我試過了，還蠻不錯的！
不過在擴充性上，還是Firefox比較好，雖然Chrome也可以擴充，但目前資源沒有比Firefox多，而且這些擴充程式在功能上也沒有像Firefox擴充套件那麼完整。

Google Chrome 下載點 http://www.google.com.tw/chrome

Mozilla Firefox—適合喜歡改造、玩擴充、玩電腦的人
Firefox的前身是Netscape，是目前我最喜歡的瀏覽器，它是開放原始碼、自由又跨平台軟體，有Windows、Linux、Mac OSX版本可以下載，目前還是Linux Fedora、Linux Ubuntu、Linux Mandriva……作業系統預設首選的網頁瀏覽器！
Firefox很有彈性，改造資源豐富，無數種佈景主題資源可以任君挑選下載美化，而且擴充程式也非常多、實用、功能又完整，可以針對個人需求，自行選用安裝。目前的擴充程式非常非常多，有擋廣告的、切換引擎成IE、抵擋網頁上不安全的Javascript、改造分頁外觀、加強下載、e-mail新郵件通知、同步書籤至遠端、增加按鈕、改造網址列……都可以自由選用。網路上還有多到不行的「Firefox必裝外掛」之類的文章。
整體而言Firefox最大的好處是改造空間大，而且很多擴充套件功能非常完整，變化性多，想把自己的瀏覽器個人化又多樣的人，Firefox是最佳選擇！

Mozilla Firefox 下載點 http://moztw.org/
給IE使用者的說明 http://support.mozilla.com/zh-TW/kb/For+Internet+Explorer+Users?style_mode=inproduct&as=u

Internet Explorer是Windows作業系統內建的瀏覽器，這是每個人都知道的。IE會有很多人使用，完全就是因為在安裝Windows後，IE就已包含在內，不需另外安裝，而且微軟(Windows的公司)當然就預設把自家的IE成為主要的瀏覽器！

已安裝Pidgin且使用Windows的人：

請到Pidgin-facebookchat計畫網站http://code.google.com/p/pidgin-facebookchat/下載「pidgin-facebookchat-1.64.exe」→下載完後直接開啟這個檔案→出現安裝程式後就照著指示安裝進去→安裝完後就照前一篇的方式一樣，把Pidgin關掉後重開→再來在Pidgin新增帳號，在「新增帳號」裡的通訊協定應該就能看到Facebook→輸入帳號密碼→試試看能不能登入，登入後要等一分鐘過後還沒斷線才算登入成功。

使用筆記：

如果成功登入後會發現明明加了那麼多朋友，為什麼只出現一點點？根據我的測試，只會出現目前在線上的人，下線的人可能也會出現，反正上線的一定會出現！

使用Linux Ubuntu 9.10，而且即時通是用Pidgin的人（這是推測！我未在9.10嘗試過）：

可以直接從套件庫安裝「pidgin-facebookchat」，安裝後需把Pidgin關掉後重開，再來在Pidgin新增帳號，在「新增帳號」裡的通訊協定應該就能看到Facebook，然後輸入帳號密碼後，應該就可以試試看能不能登入，登入後要等一分鐘過後還沒斷線才算登入成功。如果出現「伺服器停止連線」字樣的話，多試幾次或換個時間試看看。也可以在「修改帳號」裡的「進階設定」調看看。
另外我建議不要勾選「Show history in new conversations」，照我測試結果，如果勾選可能會讓沒上線的人卻顯示為上線！

使用Linux Ubuntu 9.04的人（我就在這版本安裝試驗）：

Ubuntu 9.04套件庫中的「pidgin-facebookchat」過於老舊，無法直接安裝使用，需另外到Pidgin-facebookchat計畫網站下載「pidgin-facebookchat-1.64.tar.bz2」檔案：http://code.google.com/p/pidgin-facebookchat/。

注意！不要下載pidgin-facebookchat-1.64.deb檔案！！！，因為這個檔案是針對Ubuntu 9.10製作，我之前安裝時就卡在相依性問題，無法按照正常程序裝，只能手動安裝來避開相依性驗證！！！

因為這是.tar.bz2檔，安裝方式會不同！
下載後到存放的資料夾按滑鼠右鍵選「以其他程式開啟」→出現視窗後在下方「使用自訂指令」的地方輸入「gksu file-roller」（因為要賦予管理者的權限，所以不能直接開）

→再來就會看到壓縮裡的資料夾，請把檔案解壓縮到對應的資料夾，例如：壓縮檔開啟到「/usr/lib/purple-2/」時，也要從主目錄開到「/usr/lib/purple-2/」並複製壓縮檔的所有檔案。


這些檔案複製後就手動安裝完成了！再來就要開始玩了！
弄好後就把Pidgin關掉後重開→再來在Pidgin新增帳號，在「新增帳號」裡的通訊協定應該就能看到Facebook→輸入帳號密碼→試試看能不能登入，登入後要等一分鐘過後還沒斷線才算登入成功。
如果出現「伺服器停止連線」字樣的話，多試幾次或換個時間試看看。也可以在「修改帳號」裡的「進階設定」調看看。
另外我建議不要勾選「Show history in new conversations」，照我測試結果，如果勾選可能會讓沒上線的人卻顯示為上線！

如果怎麼試都沒成功的話，可以試著更新Pidgin看看，我是更新後才成功的，我現在用的版本是Pidgin 2.6.5 ，有關更新的方法，請自行到Pidgin官網下載，如果我有空，說不定也會寫在網誌裡。

使用微軟Windows的人：

我已於2月21日在Windows裡實驗過了，相關說明請參考http://yuan817.spaces.live.com/blog/cns!A519DE06879C8B75!2071.entry

我自行測試後目前發現的問題

  1. 載入狀況實在是太不穩定！有時可以上，有時一直跳出「伺服器關閉連線」的錯誤訊息
  2. 在Pidgin開Facebook後，有時反而不能開Facebook網頁，有時又可以同時開，狀況很亂
  3. 聊天時我方、對方竟然是開不同的視窗，像是網誌上最前面的一張圖。雖然影響不大，只是用起來不習慣
  4. 我剛剛有說了，在進階設定的第一個選項建議不要勾，可能會讓好友狀況標示錯誤
